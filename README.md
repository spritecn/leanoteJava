# leanoteJava
基于leanote服务端布署相对比较难，难点主要在mongodb的资源和配置， 且官方已2年未进行维护，开设此项目

# 愿景
- 支持sqlite,mysql,postgreSql 等关系数据库，降低布署和数据理备份处理难度
- 使用java开发，便于多人合作（java群体相对多一点)
- KISS，简单易理解，易布署
- 客户端支持蚂蚊笔记原有的windows,linux,apk,ios

# 技术选型
- java8
- sparkJava,sql2o
- that's all
